Git博客模板-Hexo页面使用

2018-01-17
  • Git 系列

Hexo 官方文档

地址：https://hexo.io/zh-cn/docs/

必备软件

  • Node.js ：http://nodejs.cn/download/

  • Git ： https://git-scm.com/downloads

安装Hexo

  • npm install -g hexo-cli

  • hexo init

  • npm install

常用命令 改变的时候按顺序执行

  • hexo clean

  • hexo generate (hexo g)

  • hexo deploy (hexo d)

  • hexo server (hexo s)

  • hexo new “postName” #新建文章

  • hexo new page “pageName” #新建页面

  • hexo generate #生成静态页面至public目录

  • hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）

  • hexo deploy #将.deploy目录部署到GitHub

  • hexo help # 查看帮助

  • hexo version #查看Hexo的版本

网页添加（标签页）左上角的图标

先去制作一个ICO图标，128*128的

然后放到你的项目根目录

然后在你的页面的head标签里面添加

<link rel="shortcut icon" type="image/x-icon" href="cxp.ico" /> 这句话

然后就成功了

电脑小技巧-(右键菜单、快速启动)

怎么样给电脑右键添加自己的程序？

1.找到注册列表 （Win+R）

2.根据路径去查找你要找的位置 （HKEY_CLASSES_ROOT\directory\background\shell）

3.新建项，就是你右键显示的名字

4.新建子项command,然后修改字文件

5.修改成你程序的路径

6.给你的应用添加图标，一定要在父级菜单下建字符串值文件不能是项，而且名字必须是icon

7.然后就没有然后了，成功了，去试试吧。（PS:右键菜单，我仿佛看到多了些什么。）

怎么样快速启动软件？

1.打开%windir% （Win+R）

2.把你想快速启动的应用快捷方式拖进去就行

4.然后就没有然后了，成功了，去试试吧。（PS:Win+R,然后输入刚才文件的名称，我仿佛看到什么东西打开了。）
